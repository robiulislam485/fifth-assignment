Important FAQ

What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a webpage as a tree of objects that can be manipulated with languages like JavaScript.

How do you select an element from DOM?

To select an element from the DOM, we can use methods like document.getElementById() for a specific ID or document.querySelector() for CSS-style selectors. These methods return the first matching element from the document.

What is the event delegation in the context of of the DOM and why is it useful?

Event delegation is a technique where a single event listener is added to a parent element to manage events for its child elements. It is useful because it reduces the number of event listeners needed and dynamically handles events for elements added later.

How do you manipulate an elements attribute and styles using the DOM?

To manipulate an element's attributes,we can use methods like setAttribute() or direct property assignment (e.g., element.src = "image.jpg"). For styles, modify the style property (e.g., element.style.color = "red").
